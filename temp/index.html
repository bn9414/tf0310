Understanding the target audience

What if the dinner get random food as apposed what he wish to have? The same would happen if the website is built without understanding the need of target audience. So we study the target audience in all the angle. We experiment who is target audience? What information they seek most? What is their primary need? and so on. This is the base in which around everything is constructed

Wireframe layout

It like a black and white layout which represents the structure of your website. It shows the number of pages, what sections each page has and how the elements inside each section is organized. The main purpose of the process is to ensure that we have covered all the essential elements of web page with right way.

PSD design

It is the colored version of wireframe layout. It is high resolution image that shows how your website looks like. It makes sure that the color combination of various elements are complementing each other. Any changes the client requests will be done in this process.
